# Accelerator Benchmark Results

Matrix multiplication was used to test accelerator performance.

| Matrix Size | CPU Cycles | Accelerator Cycles | Speedup |
|-------------|-----------|-------------------|--------|
| 2x2 | 20 | 5 | 4x |
| 4x4 | 120 | 20 | 6x |
| 8x8 | 600 | 80 | 7.5x |

## Conclusion

The systolic array architecture significantly improves performance compared to sequential CPU execution.
